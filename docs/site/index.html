The Three Pillars_ □ X
OBSERVESteppable execution tree with full state inspection at every span.
REPLAYBranch from any span with an edited prompt / tool response and re-run deterministically.
COMPARESide-by-side trace diff with semantic alignment and token-level divergence.
Quick Start_ □ X
pip install -e packages/sdk-python

import stethoscope
stethoscope.attach(graph)          # one line

# run the ingestion endpoint, then:
stethoscope list-traces            # your runs, as a steppable program

Why It Looks Like 1998_ □ X
The aesthetic is deliberate: Visual Studio 6 / Windows 2000 / Borland Turbo Debugger. Density is a feature. Every action has a keyboard shortcut. It is a tool, not a chat window — it never talks to you in natural language.
MIT licensed. Built as a PRD-driven engineering project. The novel bit — deterministic replay of non-deterministic LLMs — is written up in docs/replay.md.
