# @stethoscope/desktop

Tauri 2 desktop shell — **"The Workbench"** (PRD section 8).

Scaffolded in **Phase 2** (PRD section 10): hand-rolled Win32/VS6 bevel CSS,
five-pane layout, splash screen, embedded ingestion service. Intentionally
empty until then so the vertical slice (Phase 1) lands first — "vertical
slices, not horizontal layers" (PRD section 13.1).

The aesthetic is non-negotiable: IBM Plex Mono, 8-color palette, 1px hard
bevels, no gradients/rounded corners. See PRD section 8 before writing any CSS.
